# Project skills

Skills from [ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills), installed for use with Cursor.

- **Standalone skills** (e.g. `changelog-generator`, `connect-apps`, `lead-research-assistant`, `skill-creator`, `webapp-testing`) — one folder per skill at this level.
- **Composio app automation** — 78+ app-specific skills (e.g. `github-automation`, `slack-automation`, `gmail-automation`) from the `composio-skills` set.

Cursor loads skills from `.cursor/skills/<skill-name>/` when the skill’s description matches the task.
